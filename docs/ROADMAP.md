# CRM Gold Hands — Roadmap

## ✅ Готово (MVP v1)
- Backend: Express + SQLite
- Leads: список, фильтры, статусы, карточка
- Jobs: букинг, SMS клиенту + технику
- Estimates: line items, налог, конвертация в Invoice
- Invoices: Mark Paid
- Technicians: управление
- Дизайн в стиле Workiz (синий)

## 🔄 В работе
- Calendar — месячный/дневной вид с drag-and-drop

## 📋 Очередь (по приоритету бизнеса)

### Приоритет 1 — Операционное ядро
- [ ] **Calendar** — месяц + день, блоки по часам, цвет по технику
- [ ] **Clients** — база клиентов, теги (Member/Standard/Premium/FROD), история джобов

### Приоритет 2 — Коммуникация
- [ ] **Messages** — SMS чат с клиентами (All/Unread/Requests)
- [ ] **Calls** — лог звонков (All/Missed/Voicemail)

### Приоритет 3 — Финансы и аналитика
- [ ] **Dashboard v2** — выручка по неделям, топ техники, конверсия лидов
- [ ] **Needs Attention** — просроченные лиды, нет ответа >1ч
- [ ] **Timesheets** — учёт часов техников

### Приоритет 4 — Автоматизация
- [ ] **Auto-SMS** при получении лида (Thumbtack webhook)
- [ ] **Follow-up** — если не ответили за 1ч, автоматический SMS напоминание
- [ ] **Twilio** — полная интеграция (SMS + звонки)

### Приоритет 5 — Мобильное приложение
- [ ] PWA (Progressive Web App) — установка на телефон как приложение
- [ ] Пуш-уведомления на телефон диспетчера

## 🏗️ Технический долг
- [ ] Авторизация (логин/пароль для диспетчера)
- [ ] Роли: Admin / Dispatcher / Technician
- [ ] Бэкап базы данных (ежедневно)
- [ ] Миграция SQLite → PostgreSQL (когда будет >10k записей)

# CRM Gold Hands — План разработки

## Цель
Заменить Workiz ($1000/мес) на свою CRM (~$30-50/мес)

## Функционал MVP

### 1. Получение лидов
- Webhook endpoint для Thumbtack (когда одобрят)
- Авто-SMS клиенту при получении лида (за 2-5 сек)

### 2. Панель диспетчера
- Список всех лидов (имя, телефон, тип работы, город)
- Статусы: Новый / В работе / Забронирован / Закрыт
- Кнопка "Позвонить" (клик → звонок)
- Кнопка "Добавить бронь"

### 3. Букинг
- Выбор даты и времени
- Выбор техника
- Выбор города/зоны
- Авто-SMS клиенту (подтверждение брони)
- Авто-SMS технику (детали работы)

### 4. Календарь
- Расписание по каждому технику
- Вид по дням/неделям

### 5. Техники
- Список техников
- Города/зоны каждого техника

## Стек
- **Backend:** Node.js + Express
- **База данных:** SQLite (потом PostgreSQL)
- **Frontend:** простой HTML/CSS/JS (без фреймворков)
- **SMS:** Twilio
- **Хостинг:** наш VPS (23.94.149.88)

## Стоимость
- Twilio SMS: ~$20-30/мес
- Сервер: уже есть
- **Итого: ~$20-30/мес**
